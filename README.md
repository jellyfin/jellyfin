# Jellyfin for HAOS
Jellyfin è un sistema multimediale open source che ti permette di gestire e trasmettere i tuoi contenuti multimediali.

## Installazione
1. Aggiungi il repository di ThCuba al tuo Supervisor.
2. Installa l'add-on 'Jellyfin Media Server'.
3. Configura le cartelle media nelle opzioni dell'add-on.

## Caratteristiche
- Trasmissione fluida su tutti i dispositivi.
- Supporto hardware transcoding.
- Integrazione nativa con Home Assistant.
- Gestione completa dei metadati.

## Note di configurazione
Assicurati che l'utente abbia i permessi di lettura sulle cartelle montate in `/media`.
